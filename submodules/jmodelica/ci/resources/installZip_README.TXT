=== Installation instructions ===

To install nightly builds do the following:

 1. If not already present, install the JModelica.org binary distribution or the JModelica.org SDK. 
    Both installers are available at http://www.jmodelica.org/binary or http://www.jmodelica.org/sdk 
    respectively.
 2. Replace the install directory found in the JModelica.org binary/SDK installation root directory 
    with the one in this archive.
 3. Done.
